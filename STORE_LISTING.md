# Store Listing

## English Description

Make long ChatGPT conversations easier to search, navigate, and revisit.

Features:

* Search messages within the currently loaded conversation
* Jump directly to matching messages and search results
* Bookmark important user and assistant messages for later reference
* Quickly return to saved content through the Conversation Navigator
* Keep long conversations organized and easier to manage
* Optional visual controls to reduce clutter and focus on recent content
* Local-first: no backend, no tracking, no analytics, and no conversation upload

This extension works only with content currently available on the page. It does not delete your ChatGPT account data, modify your conversation history, or upload any conversation content. All bookmarks and navigation data are stored locally in your browser.

## Chinese Description

让超长 ChatGPT 对话更容易搜索、导航和管理。

功能：

* 在当前已加载的对话内容中搜索消息
* 直接跳转到匹配结果和目标消息
* 收藏重要的用户消息和助手回复
* 通过对话导航器快速回到已收藏内容
* 让长对话更有条理、更易于管理
* 提供可选的视觉管理功能，帮助聚焦当前内容
* 本地优先：无后端、无追踪、无统计、不会上传任何对话内容

此扩展仅作用于当前页面已加载的对话内容，不会删除你的 ChatGPT 账号数据，不会修改你的对话历史，也不会上传任何对话内容。所有书签和导航数据均保存在浏览器本地。
